# 《创意编程》第五次作业——图像/交互

### 概述（实现内容）：

一款简单的绘图工具，可以提供三种功能，分为是黑笔、绘制花、以及橡皮擦。

Change Pen下设三个按钮，从左到右依次是，黑笔，特效笔（鼠标按住后光标周围会出现若干红色圆形小圈），橡皮擦。
