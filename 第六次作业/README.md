# 第六次作业：自组织与涌现性

###### 519021910096 胡沛筠

## 概述：

创建粒子系统，粒子在画面中按一定规律进行运动。

交互功能如下：

​		1）鼠标按住与松开或键盘’1‘和‘2’键 让粒子运动或停止。

  	  2）键盘’c‘和‘r’键 让粒子切换不同的运动规律。
