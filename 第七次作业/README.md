# 《创意编程》第七次作业——复杂性｜自相似与分形

###### 519021910096 胡沛筠

### 概述（实现内容）：

通过分形模拟海草和珊瑚的生长。



### 交互方式：

按下'c'、‘r'来切换是海草出现还是珊瑚出现。

鼠标点击位置海草或者珊瑚开始生长。
